## 4. E-mail Marketing

### 4.1 E-mail Lijst Segmentatie

| Segment | Criteria | Grootte (doel Y1) |
|---------|----------|-------------------|
| Prospects | Offerte aangevraagd, niet geconverteerd | 500 |
| Actieve klanten | In behandeling | 50 |
| Voormalige klanten | Project afgerond | 300 |
| Nieuwsbrief subscribers | Alleen nieuwsbrief | 1.000 |
| Partners | Aannemers, architecten | 100 |

### 4.2 E-mail Flows

**Flow 1: Lead nurture (prospects)**
- Dag 0: Welkom + gratis gids
- Dag 3: Case study relevant project type
- Dag 7: FAQ / veelgestelde vragen
- Dag 14: Testimonial + soft CTA
- Dag 30: "Nog steeds interesse?" + aanbieding

**Flow 2: Onboarding (nieuwe klanten)**
- Direct: Opdrachtbevestiging
- Dag 1: Wat kunt u verwachten
- Dag 3: Voortgang update
- Dag X: Oplevering
- Dag X+3: Feedback verzoek

**Flow 3: Reactivatie (oude klanten)**
- 6 maanden: "Nieuwe plannen?"
- 12 maanden: Nieuwjaars check-in
- Seizoensgebonden: Voorjaar/najaar campagne

### 4.3 Nieuwsbrief

**Frequentie:** Maandelijks

**Template:**
1. Hero: Spotlight project van de maand
2. Tips section: 2-3 bouw tips
3. Nieuws: Updates regelgeving/markt
4. Team: Behind the scenes
5. CTA: Offerte aanvragen

## 7. Performance Tracking

### 7.1 Attribution Model

**First-touch vs Last-touch:**
- First-touch: Waar kwam de lead vandaan?
- Last-touch: Wat triggerde conversie?

**Multi-touch attribution:**
```
Touchpoint 1: Google Ads (40% credit)
Touchpoint 2: Email nurture (20% credit)
Touchpoint 3: Website visit (10% credit)
Touchpoint 4: Retargeting ad (10% credit)
Touchpoint 5: Direct visit + conversie (20% credit)
```

### 7.2 UTM Strategie

**Format:**
```
utm_source = platform (google, facebook, linkedin, email)
utm_medium = type (cpc, organic, social, email)
utm_campaign = naam (dakkapel-q1-2026, retargeting-feb)
utm_content = variant (ad-v1, email-header-cta)
```

**Voorbeelden:**
```
?utm_source=google&utm_medium=cpc&utm_campaign=dakkapel-search-q1
?utm_source=instagram&utm_medium=social&utm_campaign=portfolio-feb
?utm_source=email&utm_medium=newsletter&utm_campaign=jan-2026
```
